# example testing module

## Installation

```bash
yarn
docker-compose up -d
```

## Usage


put all your tests in
`tests/ui`  

Run your tests

```bash
yarn start:test:ui
```

Run library tests

```bash
yarn start:test:ui-lib
```

Run all tests

```bash
yarn start:test
```

You can find all documention on [prestashop_test_lib](https://github.com/PrestaShopCorp/prestashop_test_lib#readme)
